Paso Descripción Detallada
1. Crear una cuenta en GitHub Si aún no tienes una cuenta en GitHub, lo primero que debes hacer es registrarte en su página oficial. Ve a GitHub, haz clic en "Sign up" (Registrarse) y completa el formulario con tu información (nombre, correo electrónico y contraseña). Si ya tienes una cuenta, simplemente inicia sesión con tu usuario y contraseña.
2. Crear un nuevo repositorio Una vez que hayas iniciado sesión, ve al icono de tu perfil en la parte superior derecha de la pantalla y selecciona "Tus repositorios". Luego, haz clic en el botón verde "Nuevo" para crear un repositorio desde cero. A continuación:
  • Dale un nombre a tu repositorio, como "mi-pagina-web".
  • Selecciona si deseas que el repositorio sea público (accesible para todos) o privado (solo accesible para ti y las personas que invites).
  • Marca la opción "Initialize this repository with a README" para que GitHub cree un archivo de inicio automáticamente.
  • Haz clic en "Crear repositorio".
3. Subir tu archivo index.html Una vez creado el repositorio, necesitas cargar el archivo "index.html". Sigue estos pasos:
  • En la página principal de tu repositorio, haz clic en "Add file" y selecciona "Upload files".
  • Arrastra y suelta tu archivo "index.html" o haz clic en "Choose your files" para seleccionarlo desde tu computadora.
  • Haz clic en "Commit changes" para confirmar la subida del archivo.
4. Activar GitHub Pages Para que tu archivo "index.html" sea accesible como una página web, necesitas activar GitHub Pages. Para hacerlo:
  • En la página principal de tu repositorio, haz clic en la pestaña "Settings" (Configuración).
  • Desplázate hacia abajo hasta encontrar la sección "GitHub Pages".
  • En "Source", selecciona la rama "main" (o "master", dependiendo de la configuración de tu repositorio).
  • En el menú desplegable que dice "/ (root)", asegúrate de que esté seleccionada la opción predeterminada.
  • Haz clic en "Save" para guardar los cambios.
  • GitHub te proporcionará una URL para acceder a tu página web, algo como: https://.github.io//
